
<h1>Scala stream example</h1>

author: Bruno Fitas < brunofitas@gmail.com >


<br/>
Example application that processes time series from a file
<br/>


**Input:**

`[time_in_seconds] [ratio]`

```
1355270609 1.80215
1355270621 1.80185
1355270646 1.80195
1355270702 1.80225
1355270702 1.80215
1355270829 1.80235
1355270854 1.80205
1355270868 1.80225
1355271000 1.80245
1355271023 1.80285
```


window_time => 60 seconds

**Output:**

`[time_in_seconds] [ratio] [events_in_window] [ratio_sum] [min_ratio] [max_ratio]`


<h2>Build with sbt</h2>


Test:

``` 
sbt test
```


Run:

``` 
sbt "run [path_to_file]"
```

Demo:

``` 
sbt "run data/time_series.txt"
```
